# Feature Extraction Framework: Level 2 - Move Level (The Action)
## Category: The Delta Analysis

**Description:** This level isolates the changes caused by a specific move, comparing the state BEFORE to the state AFTER.

### 1. Immediate Forcing Events
* **Capture_Type:** [None, Capture, Exchange, En_Passant] - The physical removal of enemy material.
* **Check_Status:** [None, Direct, Discovered, Double, Cross-Check] - Direct threats to the King.
* **Promotion_Result:** [None, Queen, Knight, Rook, Bishop] - Transformation of a pawn.
* **Sacrifice_Flag:** Boolean - Moving higher-value material into an attack by lower-value material for compensation.

### 2. Offensive Threat Generation
* **New_Direct_Attacks:** List/Count - Enemy pieces targeted for the first time by the moving piece.
* **Discovered_Threats:** List/Count - Threats activated by clearing a line for a sliding piece behind the moving piece.
* **Mate_Threat_Delta:** Boolean - Creation of a forced mating sequence that was not present in the previous state.
* **Tactical_Creation:** [Fork, Pin, Skewer, etc.] - New tactical motifs established by the move's final square.

### 3. Defensive Shifts
* **Reinforcement_Count:** Integer - Number of friendly pieces that gained a new defender.
* **Abandonment_List:** List - Friendly pieces or critical squares that lost their defender due to the move.
* **Interception_Flag:** Boolean - True if the move blocks an existing enemy attack ray (prophylaxis).

### 4. Positional State Changes
* **Structural_Delta:** [Opened_File, Closed_File, Created_Passer, etc.] - Permanent changes to the board's pawn/file skeleton.
* **Outpost_Status:** Boolean - True if a minor piece has successfully occupied a protected square in enemy territory.
* **Mobility_Delta:** Float - The mathematical difference (delta) in the piece's mobility compared to its previous square.

### 5. Comparative Value (The Engine Delta)
* **PV_Rank:** Integer - The move's position in Stockfish's suggested move list (PV1, PV2, etc.).
* **Centipawn_Loss:** Float - The difference in evaluation (in 1/100ths of a pawn) between this move and the best move.
* **Complexity_Spike:** Float - The change in the opponent's "branching factor." High values indicate the move made the game significantly harder for the opponent to navigate correctly.
